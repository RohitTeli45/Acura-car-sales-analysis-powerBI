# 🚗 Acura Car Sales Dashboard

## 📌 Project Overview

This Power BI dashboard provides insights into Acura car sales performance, revenue trends, customer behavior, and regional sales distribution. It helps stakeholders monitor key metrics and make data-driven business decisions through interactive visualizations.

## 🎯 Key Features

* Sales Performance Analysis
* Revenue & Profit Tracking
* Regional Sales Insights
* Customer Trend Analysis
* Interactive KPI Dashboard

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization

## 📊 Dashboard Insights

* Total Sales Overview
* Revenue Performance
* Top Performing Regions
* Customer Purchase Trends
* Key Business KPIs
